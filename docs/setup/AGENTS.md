# Setup Docs Agent Guide

## Scope
`docs/setup` captures local environment and onboarding workflows.

## Intended Topics
- local backend setup on macOS
- iPhone app local deployment prerequisites
- local network and device onboarding
- provider credential and key handling guidance

## Editing Guidance
- Prefer deterministic, step-by-step setup instructions.
- Include exact commands where possible.
- Keep setup docs aligned with scripts under `apps/ios/scripts`.
